When Do We Use A / AN?

First mention 🆕
I saw a dog in the park.
Primera vez que mencionas algo.

Solo para cosas contables en singular.

Para describir ocupaciones.

01

The Rule: Sound, Not Spelling

The choice between A and AN depends on how the next word sounds when spoken — not how it is written. Vowel sounds → AN  |  Consonant sounds → A

02

⚠ Tricky Words — Look vs Sound

Some words look like they start with a vowel but sound like a consonant, and vice versa. Always trust the sound!

✦ Starts with vowel letter but uses AN
hour /aʊər/ — H is silent!
AN hour
honest /ˈɒnɪst/ — H is silent!
AN honest person
honor /ˈɒnər/ — H is silent!
AN honor
heir /ɛər/ — H is silent!
AN heir
✦ Starts with vowel letter but uses A
university /juː.../ — sounds like "Y"!
A university
European /jʊər.../ — sounds like "Y"!
A European city
uniform /juː.../ — sounds like "Y"!
A uniform
one-way /wʌn/ — sounds like "W"!
A one-way ticket

Example Sentences

✦ Using A
She has a cat and a dog at home.
Ella tiene un gato y un perro en casa.
✦ Using AN
He is an architect and an artist.
Él es arquitecto y artista.
✦ Using A
I need a new phone. My old one is broken.
Necesito un teléfono nuevo. El viejo está roto.
✦ Using AN
It was an amazing experience!
¡Fue una experiencia increíble!
✦ Using A
Let's find a good restaurant for dinner.
Busquemos un buen restaurante para cenar.
✦ Using AN
She sent me an email about the job offer.
Ella me envió un email sobre la oferta de trabajo.
✦ Using A — tricky!
I'm studying at a university in Bogotá.
Estoy estudiando en una universidad en Bogotá. (/juː/)
✦ Using AN — tricky!
We waited an hour for the taxi.
Esperamos una hora el taxi. (H is silent!)
✖ A/AN — Negative with no
I don't have a car. / I have no car.
No tengo carro. (ambas formas correctas)
? Questions with A / AN
Is there a hospital nearby? / Do you have an idea?
¿Hay un hospital cerca? / ¿Tienes una idea?

📌 Key Notes

  • La regla es simple: A antes de sonido consonante · AN antes de sonido vocal
  • Es el SONIDO, no la letra escrita — an hour (H silenciosa) · a university (suena /juː/)
  • A / AN solo se usa con sustantivos contables en singular — nunca con plurales ni incontables.
  • Para plural: some → some books, some ideas
  • Para incontable: some o sin artículo → some water, some money, coffee
  • Para profesiones: siempre se usa a / an → She is a nurse. He is an actor.

⚠ Errores comunes:

✗ She is engineer.    ✓ She is an engineer.

✗ I have an uniform.    ✓ I have a uniform. (/juː/ sound!)

✗ It was a amazing trip.    ✓ It was an amazing trip.

✗ I want a informations.    ✓ I want some information. (uncountable!)
