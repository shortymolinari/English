There Is / There Are

Describing what exists, what's around you — Present & Past

SINGULAR There is + a/an + noun
PLURAL  There are + some/number + noun
PAST     There was / There were
00

When Do We Use It?

Existence ✓
There is a park near my house.
Para decir que algo existe o está presente.
Description 🏙
There are many restaurants here.
Para describir lo que hay en un lugar.
First mention 🆕
There is a new student in class.
Para introducir algo nuevo en la conversación.
Quantity 🔢
There are three bedrooms.
Para indicar cuántas cosas hay.
01

There Is vs There Are

THERE IS — singular (1 thing)
There is a cat on the roof.
Hay un gato en el techo.
There is a problem.
Hay un problema.
There is a supermarket nearby.
Hay un supermercado cerca.
There is some milk in the fridge.
Hay leche en la nevera. (incontable)
There is a new message for you.
Hay un nuevo mensaje para ti.
THERE ARE — plural (2+ things)
There are two cats on the roof.
Hay dos gatos en el techo.
There are some problems.
Hay algunos problemas.
There are three supermarkets.
Hay tres supermercados.
There are some eggs in the fridge.
Hay huevos en la nevera.
There are five new messages.
Hay cinco mensajes nuevos.
02

Affirmative Sentences

Structure There is / There are + a / an / some / number + noun
✦ Affirmative — singular
There is a beautiful park downtown.
Hay un parque hermoso en el centro.
✦ Affirmative — plural
There are 30 students in the class.
Hay 30 estudiantes en la clase.
✦ Affirmative — singular
There's a good coffee shop on that corner.
Hay una buena cafetería en esa esquina.
✦ Affirmative — plural
There are some interesting options.
Hay algunas opciones interesantes.
✦ Affirmative — uncountable
There is a lot of traffic today.
Hay mucho tráfico hoy.
✦ Affirmative — plural
There are many things to do in Medellín!
¡Hay muchas cosas que hacer en Medellín!
03

Negative Sentences

Structure There isn't / There aren't + a / any + noun
✖ Negative — singular
There isn't a pharmacy near here.
No hay una farmacia cerca de aquí.
✖ Negative — plural
There aren't any seats left.
No quedan asientos.
✖ Negative — singular
There isn't enough time.
No hay suficiente tiempo.
✖ Negative — plural
There aren't any good movies tonight.
No hay buenas películas esta noche.
✖ Negative — singular
There's no wifi in this area.
No hay wifi en esta zona.
✖ Negative — plural
There are no buses on Sundays.
No hay buses los domingos.
04

Yes / No Questions

Structure Is there / Are there + a / any + noun + ?
? Yes/No — singular
Is there a bank nearby?
¿Hay un banco cerca? → Yes, there is. / No, there isn't.
? Yes/No — plural
Are there any seats available?
¿Hay asientos disponibles? → Yes, there are!
? Yes/No — singular
Is there a pool in the hotel?
¿Hay piscina en el hotel? → No, there isn't.
? Yes/No — plural
Are there any good restaurants here?
¿Hay buenos restaurantes aquí? → Yes, there are many!
? Yes/No — singular
Is there a parking lot?
¿Hay un parqueadero? → Yes, there's one on the left.
? Yes/No — plural
Are there any problems with the order?
¿Hay problemas con el pedido? → No, there aren't.
05

WH Questions

Structure How many / How much / What + is there / are there + ?
◈ WH — How many
How many rooms are there?
¿Cuántos cuartos hay? → There are three.
◈ WH — How much
How much time is there?
¿Cuánto tiempo hay? → There's about an hour.
◈ WH — What
What is there to do in your city?
¿Qué hay para hacer en tu ciudad? → There's a lot!
◈ WH — Where
Where is there a good place to eat?
¿Dónde hay un buen lugar para comer?
◈ WH — How many
How many people are there at the party?
¿Cuántas personas hay en la fiesta? → About fifty.
◈ WH — What kind
What kind of shops are there nearby?
¿Qué tipo de tiendas hay cerca? → There are bakeries and cafés.
06

Past: There Was / There Were

Past Singular: There was  ·  Plural: There were  ·  Neg: There wasn't / weren't
✦ THERE WAS — singular past
There wasa cinema here before.
There wasa big earthquake in 1999.
There wasno signal on the mountain.
Was therea bus stop here before?
There wasn'ta solution at first.
✦ THERE WERE — plural past
There weremany people at the event.
There wereno flights that day.
There werethree cats in the yard.
Were thereany problems? → No, there weren't.
There weren'tenough chairs for everyone.

08

Real Conversation

Isabella busca apartamento en Medellín y habla con el arrendador Carlos. Naranja = there is/are/was/were

I
Isabella
Looking for an apartment
↔
C
Carlos
Landlord in El Poblado
I
Hi Carlos! Can you tell me about the apartment? Is there a living room?
¡Hola Carlos! ¿Me puedes hablar del apartamento? ¿Hay sala?
C
Yes! There is a big living room and a modern kitchen. There are also two bedrooms and two bathrooms.
¡Sí! Hay una sala grande y una cocina moderna. También hay dos habitaciones y dos baños.
I
That sounds great! Is there a parking space? I have a car.
¡Eso suena genial! ¿Hay parqueadero? Tengo carro.
C
Yes, there is one parking space included. And there are security cameras in the building.
Sí, hay un espacio de parqueo incluido. Y hay cámaras de seguridad en el edificio.
I
Are there any shops or restaurants nearby? I don't have a car every day.
¿Hay tiendas o restaurantes cerca? No siempre tengo el carro.
C
Absolutely! There are three supermarkets within walking distance. There is also a metro station two blocks away.
¡Por supuesto! Hay tres supermercados a distancia a pie. También hay una estación de metro a dos cuadras.
I
Perfect! Is there a gym or a pool in the building?
¡Perfecto! ¿Hay gimnasio o piscina en el edificio?
C
There isn't a pool, but there is a small gym on the 3rd floor. There are also some green areas on the rooftop.
No hay piscina, pero hay un pequeño gimnasio en el 3er piso. También hay áreas verdes en la terraza.
I
Were there any issues with the previous tenant? I just want to know.
¿Hubo algún problema con el inquilino anterior? Solo quiero saber.
C
No, there weren't any problems at all. She was a great tenant! There was only a small repair on the kitchen sink, but it's already fixed.
No, no hubo ningún problema. ¡Ella fue una excelente inquilina! Solo hubo una pequeña reparación en el lavaplatos de la cocina, pero ya está arreglado.
I
That's reassuring. Is there any flexibility with the price? 😊
Eso es tranquilizador. ¿Hay algo de flexibilidad con el precio? 😊
C
Ha! There is always a little room to negotiate. Come visit and we'll talk! 😄
¡Ja! Siempre hay un pequeño espacio para negociar. ¡Ven a visitar y hablamos!

📌 Key Notes

  • There is → 1 cosa (singular) o sustantivo incontable (water, time, money)
  • There are → 2 o más cosas (plural)
  • Contracción: There's = There is  (muy común en conversación)
  • Negativas: There isn't / There aren't any — usa any con plural
  • Preguntas: el orden se invierte → Is there / Are there
  • Pasado: There was (singular) / There were (plural) — exactamente como was/were
  • Respuestas cortas: Yes, there is. / No, there aren't. — no se dice "Yes, there's"

⚠ Error muy común:

✗ There is many people.    ✓ There are many people.

✗ Is there some coffee?    ✓ Is there any coffee? (preguntas y negativas → any)

✗ Yes, there's.    ✓ Yes, there is. (respuestas cortas no se contraen)
